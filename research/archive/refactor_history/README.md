# Refactor History Archive

These files are historical reports from the Event Alpha refactor period. They
are retained for auditability only.

Current architecture and project-health reports live in:

- `research/ARCHITECTURE_*`
- `research/PROJECT_HEALTH_*`
- `research/EVENT_ALPHA_*`

New code, tests, runbooks, and Makefile targets should use the current
architecture/project-health names. Do not add new current tooling that writes or
depends on refactor-era report names.
